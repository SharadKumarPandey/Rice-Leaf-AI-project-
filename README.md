# Rice-Leaf-AI-project-
Rice Leaf Disease Detection using Deep Learning  A computer vision project that detects and classifies rice leaf diseases using CNN and transfer learning. The model analyzes leaf images to identify infections and supports early diagnosis for farmers. Built with Python, TensorFlow, and OpenCV to improve agricultural monitoring and reduce crop loss.

----------------------------
Project Overview

This project uses deep learning and computer vision to detect and classify rice leaf diseases from images. The model analyzes leaf images and predicts the type of disease, helping farmers take early preventive actions to reduce crop loss.

The solution is built using Convolutional Neural Networks (CNN) with transfer learning for accurate image classification.

Problem Statement

Rice crops are often affected by diseases that reduce yield and quality. Manual disease detection is time-consuming and requires expert knowledge.

This project aims to build an AI-based automated disease detection system that can quickly identify infected rice leaves from images.
